// Features

Everything you need.
Nothing you don't.

Cobra strips Python down to the metal. Same syntax, zero overhead.

🐍
Python Syntax

If you know Python, you know Cobra. Indentation, def, if, while, return — identical. Only type annotations are added.

⚡
Machine Code Output

Cobra compiles to native binaries via LLVM. No interpreter, no VM, no JIT warmup — straight to metal.

🧠
Static Typing

C-style types: i8, i16, i32, i64, u8–u64, f32, f64, bool, void, ptr<T>. Caught at compile time, not runtime.

🔩
Freestanding

No libc. No stdlib. Direct syscalls. Cobra runs before any OS is loaded — perfect for kernel development.

📦
Structs & Arrays

Full struct support with member access. Arrays as typed pointers. Manual memory via mmap — you control every byte.

🎯
Multi-Target

LLVM backend targets x86-64, ARM, RISC-V, and custom ISAs. Sectum-V homebrew CPU backend in development.

// How it works

From .co to binary
in one command.

The Cobra compiler is a clean pipeline — each stage has one job. No magic, no black boxes.

lexer.py — Tokenizer
re-based tokenizer with INDENT/DEDENT support. Handles both Python-style and curly-brace blocks.
parser.py — AST Builder
Recursive descent parser. Produces a typed AST with nodes for every language construct.
checker.py — Type Checker
Validates types, resolves struct fields, catches errors before codegen.
codegen/llvm.py — IR Generator
Emits LLVM IR. SSA form with alloca/store/load. Structs, arrays, syscalls, inline asm.
llc + ld — Binary
LLVM compiles IR to object file. Static linker produces a freestanding ELF binary.
